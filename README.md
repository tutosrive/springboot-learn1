## How use it?

1. Clone this repository:

```shell
git clone https://github.com/tutosrive/springboot-learn1.git
```

2. Go to the folder (where repository was cloned)

```shell
cd springboot-learn1
```

3. Open the folder as JetBrains/Vs Code/Notepad project ... (where you want!)
4. Sync Maven Dependencies
5. Run Main Class.
